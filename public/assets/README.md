# Ảnh thương hiệu

Lưu 3 file sau vào đúng thư mục này (`frontend/public/assets/`), đúng tên:

| File | Mô tả | Dùng ở |
|------|-------|--------|
| `logo.png` | Logo hoa vàng | Topbar, trang Login |
| `avatar-nam.png` | Avatar nam (mặc định khi giới tính = NAM hoặc chưa xác định) | Avatar người dùng |
| `avatar-nu.png` | Avatar nữ (khi giới tính = NU) | Avatar người dùng |

> Tham chiếu trong code: `frontend/src/utils/brand.js`.
> Nên dùng ảnh vuông (1:1), nền trong suốt hoặc trắng. Avatar khuyến nghị ~256×256.
